\section{CLUSTERING}
The clustering component provides fine-grained routing signals for retrieval by uncovering latent structure in the medical exam questions. 

\subsection{Methodology and Implementation}
The implementation uses BERTopic \cite{grootendorst2022}. Embeddings produced by MiniLM-L12-v2 are reduced in dimensionality using UMAP \cite{mcinnes2018} and clustered using HDBSCAN \cite{campello2013}. BERTopic discovers 55 clusters alongside outlier noise points.

\subsection{Evaluation Results}
BERTopic achieved Silhouette = 0.069, and Topic Coherence $C_v = 0.5088$.
