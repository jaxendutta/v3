\section{RAG PIPELINE \& ABLATION BENCHMARK}

\subsection{Pilot Run}
A pilot evaluation using Qwen3-4B (standard) with Qwen3-Embedding-0.6B resulted in a RAG accuracy of 38\% vs a baseline of 42\% (-4pp delta). This prompted the hypothesis that standard LLMs cannot effectively use retrieved context.

\subsection{Interpretation}
The ablation reveals that the combination of a biomedical-grade embedding with a reasoning-capable LLM (Thinking-2507) is required for RAG to succeed.
